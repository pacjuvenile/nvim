## 快捷键配置

### Alt相关
|快捷键 |功能   |
| :--:  | :--:  |
| A-r   | 终端内回溯命令历史     |
| A-t   | 终端内遍历文件     |
| A-y   | ctrl+c     |
| A-p   | ctrl+v     |
| A-h   | left     |
| A-j   | down     |
| A-k   | up     |
| A-l   | right     |
| A-c   | 终端内查找并切换文件目录     |
| A-S-t   | 开启一个新的终端窗口     |
| A-S-f   | 全局查找文件     |
